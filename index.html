WELLCOME.

Epic Quest se trata de un juego en el que tienes que ir superando una serie de oleadas de enemigos. A diferencia de otros juegos aquí solo ganas puntos al pasar de nivel y se pierden cada vez que un enemigo logra pasar el mapa.

Como crear Mapas:

Se pueden crear mapas de una forma sencilla solo con archivos txt. Adjunto una plantilla para poder crear mapas: Cada - es un espacio en blanco donde no aparecerá ningún enemigo. Podemos añadir enemigos cambiando los - por: "p" -> Crearemos un pirata. Este enemigo se encargará de perseguirnos "n" -> Crearemos un npc muy rápido. "f" -> Son los enemigos mas lentos.
